Instalación de Composer

En esta sección encontrarás los pasos para instalar Composer tanto en Windows como en Mac.

Instalación en Windows

Paso 1: Descargar el instalador de Composer

Descarga el instalador de Composer para Windows desde el siguiente enlace:

Descargar Composer
Paso 2: Ejecutar el instalador

Ejecuta el instalador que acabas de descargar y sigue las instrucciones en pantalla para completar la instalación.

Paso 3: Verificar la instalación

Abre una terminal de comandos (cmd) y ejecuta el siguiente comando para verificar que Composer se ha instalado correctamente:

composer --version

Instalación en Mac

Paso 1: Descargar y ejecutar el script de instalación de Composer

Abre una terminal y ejecuta el siguiente comando para descargar e instalar Composer:

php -r "copy('https://getcomposer.org/installer', 'composer-setup.php');" && php composer-setup.php && php -r "unlink('composer-setup.php');"
Paso 2: Mover Composer a un directorio en tu PATH

Mueve Composer a un directorio en tu PATH para poder ejecutarlo globalmente:

sudo mv composer.phar /usr/local/bin/composer
Paso 3: Verificar la instalación

Para verificar que Composer se ha instalado correctamente, ejecuta el siguiente comando en tu terminal:

composer --version

Documentación adicional

Para más detalles y opciones de configuración, consulta la documentación oficial de Composer.
