Instalación de Drush

En esta sección encontrarás los pasos para instalar Drush tanto en Windows como en Mac.

Instalación en Windows

Paso 1: Instalar Composer

Asegúrate de tener Composer instalado en tu sistema. Puedes seguir las instrucciones en la sección Instalación de Composer.

Paso 2: Instalar Drush globalmente

Ejecuta el siguiente comando en tu terminal para instalar Drush globalmente usando Composer:

composer global require drush/drush
Paso 3: Verificar la instalación

Para verificar que Drush se ha instalado correctamente, ejecuta el siguiente comando en tu terminal:

drush --version

Instalación en Mac

Paso 1: Instalar Composer

Asegúrate de tener Composer instalado en tu sistema. Puedes seguir las instrucciones en la sección Instalación de Composer.

Paso 2: Instalar Drush globalmente

Ejecuta el siguiente comando en tu terminal para instalar Drush globalmente usando Composer:

composer global require drush/drush
Paso 3: Verificar la instalación

Para verificar que Drush se ha instalado correctamente, ejecuta el siguiente comando en tu terminal:

drush --version

Documentación adicional

Para más detalles y opciones de configuración, consulta la documentación oficial de Drush.
