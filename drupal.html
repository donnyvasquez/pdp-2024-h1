Instalación de Drupal

1) Instalación de Drupal en general con Drush

A continuación se describen los pasos para instalar Drupal usando Drush en Windows y Mac.

Instalación en Windows

Paso 1: Crear un nuevo proyecto de Drupal usando Composer
composer create-project drupal/recommended-project my_site_name_dir
Paso 2: Navegar al directorio del proyecto
cd my_site_name_dir
Paso 3: Ejecutar la instalación de Drupal con Drush
drush site:install

Instalación en Mac

Paso 1: Crear un nuevo proyecto de Drupal usando Composer
composer create-project drupal/recommended-project my_site_name_dir
Paso 2: Navegar al directorio del proyecto
cd my_site_name_dir
Paso 3: Ejecutar la instalación de Drupal con Drush
drush site:install

2) Instrucciones para Radix

Paso 1: Habilitar componentes
composer require drupal/components
drush en components -y
Paso 2: Descargar y habilitar el tema Radix
composer require drupal/radix
drush then radix
Paso 3: Crear un subtema usando Drush

Suponiendo que estás usando el proyecto recomendado de Drupal:

drush --include="web/themes/contrib/radix" radix:create SUBTHEME_NAME
Paso 4: Establecer el tema por defecto
drush then SUBTHEME_NAME -y; drush config-set system.theme default SUBTHEME_NAME -y
Paso 5: Instalar dependencias
cd /path/to/SUBTHEME_NAME
nvm use
npm install
Paso 6: Configurar el proxy
cp config/proxy.example.js config/proxy.js

Actualiza la variable proxy con la dirección de tu localhost en config/proxy.js (este archivo está .gitignored por defecto para evitar sobrescrituras).

Paso 7: Ejecutar el comando watch
npm run watch
